The WebWorkers API is a draft HTML5 specification for running JavaScript in the background. It is one of a number of new HTML5 features that are desireable for developers of web applications.

In this four-part series, Scott Logic developer, Jonathan Cardy, takes a detailed look at the WebWorkers API and puts it through its paces.

In the first blog post, Jonathan looks at the performance of WebWorkers across all the major browsers, in the second blog post, Jonathan looks at how WebWorkers can be combined with the new jQuery Deferred plugin, creating a concise and fluent syntax. In the third post WebWorkers are created without the need to add the to a separate JavaScript file, and finally, the fourth post re-visits jQuery, coordinating WebWorkers via fork and join.
