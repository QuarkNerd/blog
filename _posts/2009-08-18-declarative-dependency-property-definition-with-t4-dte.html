This blog post describes a technique for specifying WPF / Silverlight Dependency Properties declaritively via attributes as illustrated by the following example: [DependencyPropertyDecl("Maximum", typeof(double), 0.0)] [DependencyPropertyDecl("Minimum", typeof(double), 0.0)] public partial class RangeControl : UserControl { ... } At design-time the declarations are read via a T4 template and the required code is generated.
